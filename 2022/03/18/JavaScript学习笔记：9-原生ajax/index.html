# 原生 ajax

ajax 是一种异步通信的方法，从服务端获取数据，达到局部刷新页面的效果。
过程：

创建 XMLHttpRequest 对象；
调用 open 方法传入三个参数 请求方式(GET/POST)、url、同步异步(true/false) ;
监听 onreadystatechange 事件，当 readystate 等于 4 时返回 responseText ;
调用 send 方法传递参数。
